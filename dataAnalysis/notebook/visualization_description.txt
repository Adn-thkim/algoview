분석 데이터명 (json 파일의 key값):차트 설명
기대효과/활용법
이슈사항

[레벨별]
- 레벨별 문제 수(level_problem_name) : 현 시점까지 풀이한 문제 중 레벨별 문제 id의 개수를 의미합니다.
- 레벨별 고유한 문제 유형 개수(level_problem_type) : 레벨별로 문제 유형 id의 개수를 의미합니다.
- 레벨별 문제 유형 비율(level_problem_type_ratio) : 사람들이 레벨별 어떤 문제 유형을 많이 풀었는지 살펴보실 수 있습니다.
- 레벨별 메소드 이용수, 누적 값(level_per_function_method) : 레벨별로 어떤 메소드를 주로 사용하여 문제를 풀이했는지 살펴보실 수 있습니다.

[메소드_모듈별]
- 메소드 이용 현황, 누적 값(function_method_sort) : 현 시점까지 풀이한 코드에서 어떤 메소드를 많이 사용했는지 순위 표를 보실 수 있습니다.

[연도별]
- 연도별 문제수 (years) : 현 시점까지 풀이한 문제 중 연도별 출제문제 개수를 보실 수 있습니다.
- 연도별 문제유형 비율 (year_type):현 시점까지 풀이한 문제 중 연도별 어떤 문제유형이 많이 출제 되었는지 보실 수 있습니다.

[출제 업체별]
- 회사별 출제문제 수(companys): 현 시점까지 풀이한 문제 중 회사별 출제문제 개수를 보실 수 있습니다.
- 회사별 문제유형 (company_type): 현 시점까지 풀이한 문제 중 회사별 어떤 문제유형이 많이 출제되었는지 보실 수 있습니다.
- 회사 별 문제유형 비교 (company_ptype_radar) : 현 시점까지 풀이한 문제 중 회사별 출제문제 유형을 비교하실 수 있습니다.

[문제 유형별]
- 문제유형별 문제수(problem_type_count) : 현 시점까지 풀이한 문제 중 문제유형별 문제 id의 개수를 의미합니다.
- 문제유형별 메소드 사용횟수(problem_type_function_method) : 문제유형별로 어떤 메소드가 많이 사용되었는지 살펴보실 수 있습니다.
- 문제유형별 빌트인 함수 사용횟수(problem_type_function) : 문제유형별로 어떤 빌트인 메소드가 많이 사용되었는지 살펴보실 수 있습니다.
- 문제유형별 빌트인 함수 제외한 메소드 사용횟수(problem_type_method) : 문제유형별로 빌트인 메소드를 제외하고 어떤 메소드가 많이 사용되었는지 살펴보실 수 있습니다.
